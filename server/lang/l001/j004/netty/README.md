#   Netty

对于网络请求一般可以分为两个处理阶段，一是接收请求任务，二是处理网络请求

I/O 模型决定连接请求接收发送，并发模型决定数据处理过程

并发系统可以采用多种并发编程模型来实现，并发模型指定了系统中的线程如何通过协作来完成分配给它们的作业。
