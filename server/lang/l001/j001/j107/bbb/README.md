### Preferences

Preferences 可以保存简单的数据格式，即一行一个名字对应一个值，值只能是简单数据。常常出现在框架产品里，主要用来作为配置文件，适合配置项。

### JSON

JSON 适合用来网络传输数据，结构较为简洁，基于一个名字对应一个值，可以表达树形关系。
