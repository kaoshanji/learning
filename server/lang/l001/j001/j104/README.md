#   I/O

输入/输出，数据来源不同，处理过程不同，目的地也不同。

从外部文件把数据读取到JVM叫输入，把JVM里的数据写进外部文件叫输出，这是在堆内操作。

数据可以不被读取到JVM而直接在 OS 内存里处理，是堆外内存，由OS操作。

Linux 上所有的设备都是文件，包括网络。

在Java中，所有I/O都是通过输入/输出流来处理的，这些流使你可以按照统一的方式来处理与各种数据源之间的通信，例如文件、网络连接和内存块。
