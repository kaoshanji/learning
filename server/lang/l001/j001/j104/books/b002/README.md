#   Java编程思想 第4版

>   第18章：Java I/O 系统

对程序语言的设计者来说，创建一个好的输入/输出(I/O)系统是一项艰难的任务。

现有的大量方案已经说明了这一点，挑战似乎来自于要涵盖所有的可能性。

不仅存在各种I/O源端和想要与之通信的接收端(文件、控制台、网络链接等)，而且还需要以多种不同的方式与他们进行通信(顺序、随即存取、缓冲、二进制、按字符、按行、按字等)。

Java类库的设计者通过创建大量的类来解决这个难题。
